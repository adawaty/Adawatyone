# UX cues pulled from heyengage.io (Instarise)

Source: https://heyengage.io/ (Shopify storefront)

## What stands out

1. **Aggressive, high-contrast hero**
   - Very short, punchy headline (all caps).
   - Immediate pain framing (“conversions” > “traffic”).
   - **Single primary CTA** repeated multiple times.

2. **Motion as credibility (not decoration)**
   - Hero background video + multiple embedded videos.
   - “Movement” communicates “this brand is active / current”.

3. **Numbers as proof, above the fold**
   - Simple stat chips (e.g., CTR increase, reach, ROAS) placed directly next to the hero.
   - No long explanation; numbers do the persuasion.

4. **Social proof early + loud**
   - Brand logos strip.
   - Founder/CEO quote block.
   - Instagram account card-like proof (followers/posts/reach).

5. **Productized services**
   - “Services” shown as a grid of cards (title + image + CTA) like products.

6. **Repetition as a persuasion tool**
   - CTAs repeated (same copy, same style).
   - Multiple “Let’s talk” motifs repeated as a pattern.

## How we can borrow this (without copying the aesthetic)

- Add an **optional “Proof strip”** section on solution pages (stats + 3–6 logos) when data is available.
- Keep **one primary CTA** per page, repeated in hero + footer (already aligned with our SolutionTemplate).
- Use **motion for meaning**: marquee / magnetic CTA / subtle background motion (already aligned).
- Make Arabic CTAs **more colloquial but still business** (e.g., “احجز مكالمة” / “ابعت نطاقك”).
